📚 Key Formula Quick Reference

KNN: argmax vote over k nearest
Sigmoid: 1 / (1 + e^−z)
Entropy: −Σ pᵢ log₂(pᵢ)
Bayes: P(h|D) = P(D|h)P(h)/P(D)
SVM margin: 2 / ‖w‖
Normal eq: W = (XᵀX)⁻¹XᵀY
Bellman: V(s) = R + γ·V(s')
Q-update: Q ← Q + α[r + γ·maxQ' − Q]
Backprop: δ = (ŷ−y)·f'(z)
SMOTE: x + λ·(x_nn − x)
AdaBoost W: ½·ln((1−ε)/ε)
F1: 2·P·R / (P + R)
R²: 1 − RSS/TSS
Silhouette: (b−a)/max(a,b)
CNN out: (n−k+2p)/s + 1
Bagging: F̂ = (1/B)ΣF̂_b(x)
