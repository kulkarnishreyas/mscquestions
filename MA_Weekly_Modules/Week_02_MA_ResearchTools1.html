1. One-Way ANOVA

ANOVA (Analysis of Variance) tests whether the means of three or more groups differ significantly. It partitions total variation into between-group variation (explained by the factor) and within-group variation (random error).

Core Formulas

SST = SSB + SSW (Total = Between-group + Within-group)

SSB = Σ nᵢ(x̄ᵢ − x̄)²  where x̄ᵢ = group mean, x̄ = grand mean, nᵢ = group size
SSW = Σ Σ (xᵢⱼ − x̄ᵢ)²  (sum of squared deviations within each group)

MSB = SSB / (k − 1)  df_between = k − 1 (k = number of groups)
MSW = SSW / (N − k)  df_within = N − k (N = total observations)

F = MSB / MSW

Decision: Reject H₀ (means are equal) if F > F_critical(k−1, N−k) at α = 0.05
Fig 2.1 – ANOVA variance partitioning: SST splits into SSB (explained) and SSW (error), producing the F-statistic

Worked Case Study – CTR of Ad Formats

Case: Which ad format has the highest Click-Through Rate?

An e-commerce company tests three ad formats across 4 weeks each.

Ad FormatWeek 1Week 2Week 3Week 4Group Mean (x̄ᵢ)
Video5.2%4.8%5.5%5.1%5.150%
Banner3.1%3.4%2.9%3.2%3.150%
Text4.0%3.8%4.2%4.1%4.025%

Step-by-Step Calculation

  1. Grand Mean: x̄ = (5.15 + 3.15 + 4.025) / 3 = 4.108%
  2. SSB = 4(5.15−4.108)² + 4(3.15−4.108)² + 4(4.025−4.108)²
    = 4(1.083)² + 4(0.930)² + 4(0.007)² = 4(1.172) + 4(0.865) + 4(0.000)
    = 4.690 + 3.460 + 0.000 = ≈ 7.97
  3. SSW = [(5.2−5.15)²+(4.8−5.15)²+(5.5−5.15)²+(5.1−5.15)²]
    + [(3.1−3.15)²+(3.4−3.15)²+(2.9−3.15)²+(3.2−3.15)²]
    + [(4.0−4.025)²+(3.8−4.025)²+(4.2−4.025)²+(4.1−4.025)²]
    ≈ 0.255 + 0.145 + 0.095 = ≈ 0.495
  4. Degrees of Freedom: df_B = k−1 = 3−1 = 2; df_W = N−k = 12−3 = 9
  5. MSB = 7.97 / 2 = 3.985
  6. MSW = 0.495 / 9 = 0.055
  7. F = MSB / MSW = 3.985 / 0.055 = F ≈ 72.5
  8. Decision: F_critical(2, 9) at α=0.05 = 4.26. Since 72.5 ≫ 4.26 → Reject H₀!

Post-Hoc Testing: Tukey HSD

When ANOVA rejects H₀, we know at least one pair of means differs. Tukey's Honestly Significant Difference (HSD) test identifies which specific pairs are different while controlling the family-wise error rate.

HSD = q × √(MSW / n)
where q = Studentised range statistic, n = group size

If |x̄ᵢ − x̄ⱼ| > HSD → that pair of groups differs significantly.

Two-Way ANOVA

Two-Way ANOVA tests main effects of TWO factors AND their interaction effect simultaneously. An interaction occurs when the effect of one factor depends on the level of the other.
Case: Restaurant Satisfaction

Factor A: Menu Type (Traditional vs Fusion)  |  Factor B: Price Range (Budget vs Premium)

Budget PricePremium Price
TraditionalSatisfaction: 6.5/10Satisfaction: 7.0/10
FusionSatisfaction: 6.8/10Satisfaction: 8.9/10

Interaction Effect: Fusion food at premium price yields 8.9 satisfaction — far more than the sum of individual effects. Customers expect (and receive) a premium experience from fusion cuisine at higher price points.

2. Factor Analysis

Factor Analysis is a dimensionality reduction technique that identifies latent (hidden) factors underlying a set of observed, correlated variables. Instead of analysing 20 survey variables separately, we might find they reduce to 3–4 underlying dimensions.

EFA – Exploratory

Used when you don't know the factor structure in advance. The algorithm finds factors from the data. Common in early marketing research.

CFA – Confirmatory

Used when you have a theory about factor structure. Tests whether observed data fits a pre-specified model. Common in scale validation.

5-Step Process

  1. Collect & standardise data: Survey N respondents on p variables (e.g., 500 customers on 10 brand attributes)
  2. Test suitability: KMO statistic > 0.5 (ideally > 0.7); Bartlett's test of sphericity p < 0.05
  3. Extract factors: Use Principal Component Analysis (PCA) — the most common extraction technique — which transforms correlated variables into a set of uncorrelated principal components, retaining only the most significant ones. Retain factors with eigenvalue > 1 (Kaiser criterion) OR use the scree-plot elbow.
  4. Rotate factors: Orthogonal rotation (e.g., Varimax — produces uncorrelated factors) or Oblique rotation (e.g., Promax — allows correlated factors)
  5. Interpret loadings: |loading| > 0.5 = strong association; name each factor based on its high-loading variables

4. Multidimensional Scaling (MDS)

MDS is a dimensionality reduction technique that visualises similarity/dissimilarity data in 2D or 3D space. Given a matrix of pairwise distances or dissimilarities between brands, MDS finds coordinates that best preserve those distances in a low-dimensional map.

Goodness of Fit Metrics

Stress value = √[Σ(d_ij − δ_ij)² / Σδ_ij²]
where d_ij = observed distance, δ_ij = reproduced distance in MDS map

Interpretation:
Stress = 0.00 → Perfect fit (ideal)
Stress < 0.05 → Excellent fit
Stress 0.05–0.10 → Good fit (acceptable)
Stress 0.10–0.20 → Fair fit (interpret with caution)
Stress > 0.20 → Poor fit (do not use)

R² > 0.90 is required for acceptable MDS solution

5-Step Process

  1. Collect similarity/dissimilarity data: Ask respondents to rate brand pairs on similarity (1–9 scale) or measure inter-brand distances
  2. Build distance matrix: Convert similarity ratings to dissimilarities (dissimilarity = 10 − similarity)
  3. Run MDS algorithm: Specify number of dimensions (usually 2); metric MDS for ratio/interval data, non-metric for ordinal
  4. Assess fit: Check stress value and R². If stress > 0.15, consider adding a dimension
  5. Interpret the perceptual map: Label axes based on which brands appear at extremes; identify gaps (opportunities) and clusters (competition)

Summary & Exam Tips

ANOVA Formula Chain

SST = SSB + SSW → MSB = SSB/(k−1) → MSW = SSW/(N−k) → F = MSB/MSW

ANOVA Decision Rule

Reject H₀ if F > F_critical (table value). Post-hoc: use Tukey HSD to find which pairs differ.

Factor Analysis Thresholds

KMO > 0.6 = acceptable; Bartlett p < 0.05; eigenvalue > 1 to retain; loading > 0.5 = strong.

Cluster Validation

Silhouette > 0.5 = good; Davies-Bouldin: lower = better; elbow = optimal k for WCSS.

MDS Stress Values

<0.05 = excellent; 0.05–0.10 = good; 0.10–0.20 = fair; >0.20 = poor. R² > 0.9 needed.

Rotation Types

Varimax = orthogonal (uncorrelated factors). Promax/Oblimin = oblique (correlated factors allowed).

DBSCAN vs K-Means

K-Means: spherical clusters, need to specify k. DBSCAN: arbitrary shapes, handles outliers, no k needed.

Two-Way ANOVA

Tests: main effect A + main effect B + A×B interaction. Interaction = effect of A depends on B.

Common Exam Pitfalls
  • df error: df_between = k−1 (not k), df_within = N−k (not N−1)
  • F-statistic direction: Always MSB/MSW (between ÷ within), never reversed
  • KMO confusion: KMO < 0.5 means do NOT proceed with factor analysis
  • Eigenvalue rule: Retain factors with eigenvalue > 1 (not ≥ 1, the scree plot elbow may differ)
  • MDS stress: Lower stress = better fit. Stress of 0.25 = poor (do NOT interpret the map)
  • DBSCAN does NOT require specifying k — this is its key advantage over K-Means
  • Silhouette Score range: −1 to +1 (not 0 to 1). Negative score = point in wrong cluster

NP1A one-way ANOVA compares the mean click-through rate across k = 4 advertising channels. What is the between-groups degrees of freedom (dfbetween)?
df_between = k − 1 = 4 − 1 = 3
NP2The same study has N = 24 total observations across the k = 4 channels. What is the within-groups degrees of freedom (dfwithin)?
df_within = N − k = 24 − 4 = 20
(Check: df_total = N − 1 = 23 = df_between + df_within = 3 + 20.)
NP3The between-groups sum of squares is SSB = 2,460 with dfbetween = 3. Compute the mean square between groups (MSB).
MSB = SSB / df_between = 2,460 / 3 = 820
NP4From the ANOVA, MSB = 820 and the mean square within (MSW) = 164. Compute the F-statistic.
F = MSB / MSW = 820 / 164 = 5.0
A larger F means group means differ more relative to within-group noise — stronger evidence against H₀.
NP5Four equal-sized groups (n = 5 each) have means of 12, 15, 18, and 11. What is the grand mean of all observations?
Equal n ⇒ grand mean = average of the group means X̄ = (12 + 15 + 18 + 11) / 4 = 56 / 4 = 14
NP6In factor analysis, a variable loads on three factors with loadings 0.72, 0.55, and 0.40. The communality is the sum of the squared loadings. Compute the communality (2 dp).
Communality = Σ (loading)² = 0.72² + 0.55² + 0.40² = 0.5184 + 0.3025 + 0.1600 = 0.9809 ≈ 0.98
About 98% of this variable's variance is explained by the three retained factors.
NP7A factor has an eigenvalue of 3.6 in an analysis of 8 standardised variables. What percentage of total variance does this factor explain?  (% = eigenvalue / #variables × 100)
% variance = eigenvalue / number of variables × 100 = 3.6 / 8 × 100 = 45%
Each standardised variable contributes 1 unit of variance, so total variance = 8.
NP8In an MDS study, respondents rate the similarity of every possible pair of 9 brands. How many distinct brand pairs must be evaluated?  (pairs = n(n−1)/2)
pairs = n(n − 1) / 2 = 9 × 8 / 2 = 72 / 2 = 36
NP9A two-way (factorial) ANOVA crosses Factor A with 3 levels and Factor B with 4 levels. What is the degrees of freedom for the interaction term (A×B)?  (df = (a−1)(b−1))
df_interaction = (a − 1)(b − 1) = (3 − 1)(4 − 1) = 2 × 3 = 6
NP10A K-Means solution yields an average silhouette score of 0.68 using S = (b−a)/max(a,b). How should this be interpreted?
Silhouette ranges from −1 to +1. Values near +1 mean tight, well-separated clusters; near 0 means overlapping; negative means likely mis-assignment. 0.68 indicates reasonably good, well-separated clustering.

Q1 · 2 marks A one-way ANOVA gives SSB = 1,200 with dfbetween = 2, and SSW = 2,400 with dfwithin = 24. What is the F-statistic?
F is the ratio of mean squares, each being a sum of squares divided by its df.
MSB = SSB / df_between = 1,200 / 2 = 600 MSW = SSW / df_within = 2,400 / 24 = 100 F = MSB / MSW = 600 / 100 = 6.0
0.5 reverses the ratio (MSW/MSB); 60 forgets to divide MSW by its df.
Q2 · 2 marks In a factor analysis, one variable loads on three retained factors with loadings 0.6, 0.5 and 0.4. What is its communality (2 dp)?
Communality = the sum of the squared loadings.
h² = 0.6² + 0.5² + 0.4² = 0.36 + 0.25 + 0.16 = 0.77
About 77% of this variable's variance is captured by the three factors. 1.50 wrongly sums the raw loadings.
Q3 · 2 marks A factor analysis of 9 standardised variables produces a factor with an eigenvalue of 4.5. What percentage of total variance does this factor explain?
Each standardised variable carries 1 unit of variance, so total variance = number of variables.
% variance = eigenvalue / #variables × 100 = 4.5 / 9 × 100 = 50%
45% wrongly divides by 10; 20% inverts the ratio.
Q4 · 2 marks A manager wants to know whether mean customer-satisfaction scores differ across four store formats (flagship, mall, high-street, kiosk). Which technique is the appropriate choice, and what is its null hypothesis?
Comparing the means of three or more groups on one metric is the job of one-way ANOVA. Its null is that all group means are equal (μ₁ = μ₂ = μ₃ = μ₄); rejecting it says at least one format differs. Factor analysis reduces variables, cluster analysis groups cases, and MDS maps perceived distances — none tests group-mean equality.
Q5 · 2 marks A researcher has 30 attitudinal survey items and suspects they really measure only a few underlying dimensions (e.g. "value", "trust", "convenience"). Which technique should be used, and why?
Factor analysis is a data-reduction technique: it collapses many inter-correlated variables into a smaller set of latent factors. Cluster analysis groups cases (respondents), not variables; ANOVA compares means; MDS maps perceived similarity/distance between objects.
Q6 · 2 marks A K-Means run returns an average silhouette score of 0.71. What does this indicate about the clustering quality?
The silhouette score ranges from −1 to +1. Values near +1 mean tight, well-separated clusters; near 0 means overlap; negative means likely mis-assignment. 0.71 indicates strong, well-separated clustering. Scores can legitimately exceed 0.5, so option D is false.
