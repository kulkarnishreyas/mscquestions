📌 Recommended Study Strategy

  1. Weekly Study: Open the week's HTML alongside the PDF. Read explanations, study diagrams, work through examples.
  2. Self-Test: Take the 14-15 question quiz at the end of each week — instant feedback (green = correct, red = wrong).
  3. Review Summary Cards: Each week has a "Summary & Exam Tips" section with key takeaways & common pitfalls.
  4. Cross-Reference: Concepts repeat — Week 3 stats appears in Week 6 ML and Week 8 VaR. Build connections.
  5. Practical: Run the Python/Excel code samples on real data (try Yahoo Finance via yfinance).
