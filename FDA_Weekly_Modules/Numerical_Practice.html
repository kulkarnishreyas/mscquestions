Week 2 — Financial Ratios

Liquidity, leverage & profitability ratios · the XYZ Corporation case · profit margins

Q2.1 — Liquidity ratios

A firm reports Current Assets = ₹200,000, Inventory = ₹50,000, Current Liabilities = ₹120,000.

Current Ratio = CA / CL Quick Ratio = (CA − Inventory) / CL

Solution

Current Ratio = 200,000 / 120,000 = 1.67 Quick Ratio = (200,000 − 50,000) / 120,000 = 150,000/120,000 = 1.25

Q2.2 — Leverage & return ratios

Total Liabilities = ₹300,000, Shareholders' Equity = ₹500,000, Total Assets = ₹800,000, Net Income = ₹100,000.

Solution

Debt-to-Equity = 300,000 / 500,000 = 0.60 Debt Ratio = 300,000 / 800,000 = 0.375 (37.5% of assets financed by debt) ROE = 100,000 / 500,000 × 100 = 20% Equity Ratio = 500,000 / 800,000 = 0.625

Q2.3 — Case study: XYZ Corporation

From XYZ Corp's balance sheet: Current Assets = ₹170,000 (incl. Inventory ₹40,000), Current Liabilities = ₹90,000, Total Liabilities = ₹240,000, Total Assets = ₹400,000, Shareholders' Equity = ₹160,000, Net Income = ₹40,000.

Solution

Current Ratio = 170,000 / 90,000 = 1.89 Quick Ratio = (170,000 − 40,000) / 90,000 = 1.44 ROE = 40,000 / 160,000 × 100 = 25% Working Capital = 170,000 − 90,000 = ₹80,000

Also: Debt-to-Equity = 240/160 = 1.5; Debt Ratio = 240/400 = 0.6; Asset Turnover = 500,000/400,000 = 1.25; ROA = 40,000/400,000 = 10%; Equity Ratio = 0.4.

Q2.4 — Profit margins

Revenue = ₹40,00,000, COGS = ₹25,00,000, Operating Income = ₹8,00,000, Net Income = ₹5,00,000.

Solution

Gross Profit = 40,00,000 − 25,00,000 = 15,00,000 Gross Margin = 15,00,000 / 40,00,000 = 37.5% Operating = 8,00,000 / 40,00,000 = 20% Net Margin = 5,00,000 / 40,00,000 = 12.5%

Week 3 — Descriptive Statistics & Regression

Central tendency · dispersion · hypothesis testing · simple linear regression

Q3.1 — Mean & median of fund returns

Weekly returns of five funds: 3%, 5%, −2%, 8%, 3%.

Solution

Mean = (3 + 5 − 2 + 8 + 3)/5 = 17/5 = 3.4% Sorted = −2, 3, 3, 5, 8 → Median = 3% (middle value) Mode = 3% (appears twice)

Q3.2 — Variance & standard deviation

Returns: 5%, −3%, 7%, 10%, 2%. The mean is 4.2%. Compute the (population) variance and standard deviation.

σ² = (1/n) Σ (xᵢ − μ)² σ = √σ²

Solution

deviations: 0.8, −7.2, 2.8, 5.8, −2.2 squares: 0.64 + 51.84 + 7.84 + 33.64 + 4.84 = 98.8 σ² = 98.8 / 5 = 19.76 σ = √19.76 = 4.44%

Q3.3 — Hypothesis test decision

You test whether a strategy beats an 8% benchmark. The test returns a p-value of 0.02 at significance level α = 0.05. What is the decision?

Solution

Since p (0.02) < α (0.05), reject H₀. There is statistically significant evidence the strategy outperforms the 8% benchmark.

Q3.4 — Simple linear regression

Advertising X (₹'000) vs sales Y (₹'000): (10,50), (15,65), (20,80), (25,95), (30,110). The relationship is perfectly linear.

Y = β₀ + β₁X

Solution

β₁ = ΔY/ΔX = (65−50)/(15−10) = 15/5 = 3 β₀ = 50 − 3×10 = 20 → Y = 20 + 3X At X = 25: Y = 20 + 3×25 = 20 + 75 = ₹95 ('000)

Interpretation: every extra ₹1,000 of advertising lifts sales by ₹3,000.

Week 4 — Time Value of Money

Present/future value · compounding frequency · perpetuity · weighted return

Q4.1 — Discount-rate sensitivity

Find the present value of ₹15,000 received in 8 years at discount rates of 3% and 7%.

PV = FV / (1 + r)ⁿ

Solution

PV(3%) = 15,000 / 1.03⁸ = 15,000 / 1.2668 ≈ ₹11,842 PV(7%) = 15,000 / 1.07⁸ = 15,000 / 1.7182 ≈ ₹8,730

A higher discount rate gives a lower present value. (The PDF rounds these to ₹11,850.27 and ₹8,736.51.)

Q4.2 — Future value & compounding frequency

Invest ₹5,000 at 6% for 5 years. Compare annual vs quarterly compounding.

FV = PV · (1 + r/m)^(n·m)

Solution

Annual = 5,000 × 1.06⁵ = ₹6,691.13 Quarterly = 5,000 × (1 + 0.06/4)^20 = 5,000 × 1.015^20 ≈ ₹6,734

More frequent compounding earns a little more. (The PDF states the quarterly figure as ₹6,744.25.)

Q4.3 — Present value of a perpetuity

A preferred share pays a fixed dividend of ₹500 forever; the discount rate is 10%. What is its present value?

PV = C / r

Solution

PV = 500 / 0.10 = ₹5,000

Q4.4 — Weighted portfolio return

A portfolio holds four stocks:

= 0.40×8 + 0.30×10 + 0.20×6 + 0.10×12 = 3.2 + 3.0 + 1.2 + 1.2 = 8.6%

Week 5 — Moving Averages

Simple & weighted moving averages on a price series

Q5.1 — 3-period Simple Moving Average

Series: 10, 20, 30, 40, 50. Compute the 3-period SMA at period 3 and period 5.

Solution

P3 = (10+20+30)/3 = 20 P4 = (20+30+40)/3 = 30 P5 = (30+40+50)/3 = 40

Q5.2 — Weighted Moving Average

Same series. Use weights 3 (most recent), 2, 1 (oldest). Compute the 3-period WMA at period 3 (window 10, 20, 30).

WMA = (3·newest + 2·mid + 1·oldest) / (3+2+1)

Solution

= (3×30 + 2×20 + 1×10) / 6 = (90 + 40 + 10) / 6 = 140/6 = 23.33

The WMA (23.33) sits above the SMA (20) because it weights the larger, most-recent value more heavily.

Week 6 — Predictive Analytics & ML

Expected monetary value · accuracy · logistic default model

Q6.1 — Expected Monetary Value (investment decision)

Project A: 60% chance of ₹100,000 gain, 40% chance of ₹50,000 loss.
Project B: 90% chance of ₹30,000 gain, 10% chance of ₹10,000 loss.

EMV = Σ (outcome × probability)

Which project should be chosen?

Solution

EMV(A) = 0.6×100,000 + 0.4×(−50,000) = 60,000 − 20,000 = ₹40,000 EMV(B) = 0.9×30,000 + 0.1×(−10,000) = 27,000 − 1,000 = ₹26,000

Project A has the higher expected value → choose A.

Q6.2 — Classification accuracy

A model makes 930 correct predictions out of 1,000. What is its accuracy?

Solution

Accuracy = (TP+TN)/total = 930 / 1,000 = 0.93 = 93%

Q6.3 — Logistic default model

The PDF's loan-default model is P(Default) = σ(2.5 − 0.02·CreditScore + 0.1·DebtRatio). For an applicant with CreditScore = 300 and DebtRatio = 50:

z = 2.5 − 0.02·CS + 0.1·DR P = 1/(1 + e^−z)

Solution

z = 2.5 − 0.02×300 + 0.1×50 = 2.5 − 6 + 5 = 1.5 P = 1/(1 + e^−1.5) = 1/(1 + 0.2231) = 0.818

≈ 82% default probability → a high-risk applicant.

Week 8 — Risk Management

Parametric VaR · market-risk sensitivity

Q8.1 — Parametric (variance–covariance) VaR

At 99% confidence the Z-score is 2.33. A portfolio's daily standard deviation is σ = 2.5%, horizon t = 1 day. Compute the 1-day VaR (as a %).

VaR = Z · σ · √t

Solution

VaR = 2.33 × 2.5% × √1 = 5.825%

At 99% confidence the daily loss is not expected to exceed ≈5.83% (vs 3.30% at 95% with σ=2%).

Q8.2 — Interest-rate sensitivity

A $1,000,000 portfolio holds 40% in equities and the rest in rate-sensitive assets. An interest-rate shift changes the non-equity portion by 1%. What is the dollar change from the rate move?

Δ = rate_sensitivity × portfolio_value × (1 − equity_weight)

Solution

= 0.01 × 1,000,000 × (1 − 0.40) = 0.01 × 1,000,000 × 0.60 = $6,000

Week 9 — Portfolio Performance

Sharpe / Treynor / Jensen's alpha · CAPM

Q9.1 — Treynor ratio & Jensen's alpha

Portfolio returns average 14%; the benchmark averages 12.8%. Risk-free rate R_f = 3%, beta β = 1.2. (So mean excess return = 14% − 3% = 11%.)

Treynor = (R_p − R_f) / β Alpha = R_p − R_benchmark

Solution

Treynor = (0.14 − 0.03) / 1.2 = 0.11 / 1.2 = 0.0917 Alpha = 0.14 − 0.128 = 0.012 (1.2% over benchmark) Sharpe = 0.11 / σ = 0.11 / 0.0283 ≈ 3.89

Q9.2 — CAPM expected return

Risk-free rate 3%, market return 10%, stock beta β = 1.2. What return does CAPM expect?

E(R) = R_f + β · (R_m − R_f)

Solution

E(R) = 3% + 1.2 × (10% − 3%) = 3% + 1.2×7% = 3% + 8.4% = 11.4%

This benchmark return is exactly what Jensen's alpha measures performance against.

Week 10 — Algorithmic Trading

Moving-average crossover on a real price series

Q10.1 — First 3-day moving average

Closing prices for the first three days are 296.24, 298.85, 299.74. Compute the 3-day simple moving average at day 3.

Solution

= (296.24 + 298.85 + 299.74) / 3 = 894.83 / 3 = 298.28

A buy signal fires when this short MA crosses above the longer (e.g. 5-day or 200-day) MA — the golden cross.
