1️⃣ Modern Portfolio Theory (MPT)

Markowitz, 1952: diversifying across uncorrelated assets reduces risk without sacrificing return.

Core Principles

🌐 Diversification

Reduces unsystematic risk

📈 Efficient Frontier

Optimal risk-return portfolios

📐 Mean-Variance

Allocate weights based on E(R) and σ

🎯 Market Efficiency

Prices reflect info — beating market needs extra risk

Two Types of Risk

TypeDiversifiable?Examples
SystematicNoRecession, war, inflation
UnsystematicYesCompany management, sector issues

Assumptions of MPT

  • Rational investors maximising utility
  • Normally distributed returns
  • Markets efficient
  • Stable correlations
  • Single-period horizon

Limitations

  • Oversimplifies risk (focuses only on mean/variance)
  • Correlations spike during crises
  • Real investors irrational
  • Past data ≠ future

Python Optimization Snippet

import numpy as np, pandas as pd from scipy.optimize import minimize returns = pd.read_csv('returns.csv').pct_change().dropna() mu = returns.mean() cov = returns.cov() def neg_sharpe(w): pr = np.dot(w, mu) pvol = np.sqrt(w.T @ cov @ w) return -pr/pvol n = len(mu) init = np.ones(n)/n bounds = [(0, 1)] * n constraints = {'type':'eq', 'fun': lambda x: x.sum()-1} opt = minimize(neg_sharpe, init, method='SLSQP', bounds=bounds, constraints=constraints) print("Optimal weights:", opt.x)

2️⃣ Portfolio Optimization Techniques

Mean-Variance Formulas

E(Rp) = Σ wᵢ E(Rᵢ)   |   σp² = ΣᵢΣⱼ wᵢ wⱼ σᵢⱼ

Common Constraints

  • Budget: Σ wᵢ = 1
  • No short selling: wᵢ ≥ 0
  • Risk tolerance: σp ≤ target
  • Position limits: wᵢ ≤ max

Sharpe Maximisation

max (E(Rp) − Rf) / σp   subject to Σwᵢ=1, wᵢ ≥ 0

3️⃣ Asset Allocation Strategies

Factors Influencing Allocation

  • Market Conditions — rate cycles, inflation
  • Financial Situation — income, debt
  • Diversification — across asset classes & geographies
  • Tax & Regulation — capital gains, tax-advantaged accounts

Python — VaR & Monte Carlo

import numpy as np # Historical VaR (95%) returns = np.random.normal(0.001, 0.02, 1000) VaR = np.percentile(returns, 5) print(f"95% VaR: ${-VaR * 1_000_000:.0f}") # Monte Carlo simulation def monte_carlo(initial, mu, sigma, days=252, sims=1000): paths = np.zeros((days, sims)) for i in range(sims): prices = [initial] for _ in range(days-1): shock = np.random.normal((mu - 0.5*sigma**2)/days, sigma/np.sqrt(days)) prices.append(prices[-1] * np.exp(shock)) paths[:, i] = prices return paths

Evaluating Effectiveness

  • Drawdown — peak-to-trough loss
  • Sharpe & Sortino — risk-adjusted
  • Win/Loss — consistency
  • Backtesting + Forward testing

Real-World Applications

  • Pension funds — LDI strategies (CalPERS)
  • Target-Date Funds — auto-rebalance with age (Vanguard)
  • Hedging in volatility — put options, inverse ETFs (ProShares Short S&P500 in 2008)
  • Dynamic allocation — Ray Dalio's All Weather Portfolio

5️⃣ Performance Measurement

Key Metrics

MetricFormulaHigher = Better?
Sharpe Ratio(Rp − Rf) / σpYes
Treynor Ratio(Rp − Rf) / βpYes
AlphaRp − [Rf + βp(Rm − Rf)]Yes (positive)
Information RatioAvg excess / SD of excessYes
Max Drawdown(Trough − Peak) / PeakLower (negative) is worse

Python Example

import numpy as np returns = np.array([0.12,0.18,0.14,0.10,0.16]) market = np.array([0.10,0.15,0.12,0.08,0.14]) rf = 0.03 excess = returns - rf sharpe = excess.mean() / returns.std() treynor = excess.mean() / 1.2 # beta=1.2 assumed alpha = returns.mean() - (rf + 1.2*(market.mean()-rf)) cum = np.cumprod(1 + returns) peak = np.maximum.accumulate(cum) drawdown = (cum - peak)/peak max_dd = drawdown.min() print(f"Sharpe {sharpe:.2f}, Treynor {treynor:.4f}, Alpha {alpha:.4f}, Max DD {max_dd:.2%}")

Why Historical Analysis?

  • Benchmark vs S&P 500 / Nifty
  • Risk-adjusted returns across periods
  • Evaluate manager skill
  • Data-driven future decisions

6️⃣ Excel Portfolio Optimization

Step-by-Step

  1. Collect historical prices (Yahoo Finance) in columns by asset
  2. Calculate daily returns: =(B3-B2)/B2
  3. Expected portfolio return: =SUMPRODUCT(weights, mean_returns)
  4. Portfolio variance/SD: variance-covariance matrix via MMULT
  5. Run Solver:
    • Objective: max Sharpe (or min variance)
    • Variable cells: weights
    • Constraints: sum(weights)=1, weights ≥ 0
  6. Generate Efficient Frontier: loop through many weight combos, plot scatter (Risk, Return)

📝 Summary & Exam Tips

🎯 Key Takeaways

  • MPT: diversify for optimal risk-return; Efficient Frontier shows best portfolios
  • Risk: systematic (can't diversify), unsystematic (can)
  • Mean-Variance: maximise E(Rp) at given σp, or vice versa
  • Strategies: Conservative → Balanced → Growth → Aggressive → Dynamic
  • Risk Mgmt: Diversify, Allocate (strategic+tactical), Hedge, VaR/MC, Rebalance
  • Performance: Sharpe (total risk), Treynor (β), Alpha (excess vs benchmark), Info Ratio (consistency), Max Drawdown
  • Excel Solver maximises Sharpe or minimises σ under constraints (Σw=1)
📌 Pitfalls:
  • Treating MPT correlations as static — they spike in crashes
  • Over-diversification dilutes winners
  • Ignoring tail risk in optimisation
  • Backtest only — also forward test
