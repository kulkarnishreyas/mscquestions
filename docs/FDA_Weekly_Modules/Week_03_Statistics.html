1️⃣ Descriptive Statistics Overview

Descriptive statistics summarise data to reveal patterns. Four families of measures cover most analyses:

📍 Central Tendency

Mean, Median, Mode — the "typical" value.

📏 Variability

Range, Variance, SD, IQR — how spread out the data is.

📐 Distribution Shape

Skewness (asymmetry) and Kurtosis (tail heaviness).

🔗 Relationships

Covariance, Correlation — how variables move together.

Quick example — fund weekly returns: 3%, 5%, -2%, 8%, 3%

  • Mean = (3+5−2+8+3)/5 = 3.4%
  • Median = 3% (sorted: -2, 3, 3, 5, 8 → middle)
  • Mode = 3% (appears twice)

2️⃣ Measures of Central Tendency

Mean = (Σ xᵢ) / n   |   Median = middle value when sorted   |   Mode = most frequent

Three-Stock Comparison

StockMeanMedianMode
A7.4%7%7%
B6.2%6%6%
C9.8%12%15%

Stock C has the highest mean but also the widest spread (volatile). Stock B is most stable.

When to use what?

📊 Mean

Best for symmetric data. Sensitive to outliers — extreme values pull it.

📍 Median

Robust to outliers/skewness. Best for income, real estate prices.

🔁 Mode

For categorical/repeated data. Trading: most-frequent price level (support/resistance).

📌 Limitations: Central tendency alone doesn't tell you about spread. Two stocks with same mean return (8%) can have wildly different risk profiles.

3️⃣ Measures of Dispersion

Core Measures

MeasureFormulaNotes
RangeMax − MinSimplest, ignores distribution
Variance (σ²)Σ(xᵢ − μ)² / NSquared units — hard to interpret
Standard Deviation (σ)√VarianceSame units as data — most popular
IQRQ3 − Q1Middle 50% — robust to outliers

Worked Example — returns: 5%, -3%, 7%, 10%, 2%

  • μ = (5 − 3 + 7 + 10 + 2)/5 = 4.2%
  • Squared deviations: 0.64, 51.84, 7.84, 33.64, 4.84
  • σ² = 98.8/5 = 19.76%²
  • σ = √19.76 ≈ 4.44%
  • Range = 10 − (−3) = 13%

Sharpe Ratio

Sharpe = (Portfolio Return − Risk-Free Rate) / σ

Compares return per unit of risk. Higher is better. Used to compare investments fairly.

📊 Three stocks compared:
  • Stock A: Mean 7%, SD 4.5% — balanced
  • Stock B: Mean 6%, SD 2.8% — stable (lowest risk)
  • Stock C: Mean 9%, SD 6.0% — high reward, high risk
Risk-averse investors prefer B; aggressive investors lean to C.

Distribution Shape

↗️ Positive Skew

Long right tail. Most values low, a few large. E.g., income.

↖️ Negative Skew

Long left tail. Most values high, a few small.

📈 High Kurtosis

Heavy tails — more outliers than normal distribution.

4️⃣ Inferential Statistics

Inferential statistics draws conclusions about a population from a sample. Crucial in finance where you can't observe every transaction.

Descriptive vs Inferential

AspectDescriptiveInferential
ScopeDataset itselfSample → Population
GoalDescribeGeneralize, predict
TechniquesCharts, summary statsHypothesis tests, regression, ANOVA
OutputSummary numbersConclusions with confidence levels

Key Inferential Methods in Finance

📈 Regression

Simple (1 predictor) or Multiple. E.g., predict stock from inflation + rates.

⏳ Time Series (ARIMA)

Forecast future from historical sequence.

🎲 Monte Carlo

Simulate 1000s of scenarios for portfolio risk.

🧪 Hypothesis Tests

Test strategy effectiveness, return differences.

📊 Confidence Intervals

Range estimate (e.g., 95% CI for mean return).

⚠️ Value at Risk (VaR)

Potential loss in extreme adverse cases.

Sample Size Matters

  • Larger sample → smaller margin of error, narrower CIs, higher statistical power
  • Small sample → wider intervals (2%–15%), risk of Type II errors
  • Balance precision vs cost — large samples are expensive

5️⃣ Hypothesis Testing

Setting Up Hypotheses

H₀ (Null)

Default — no effect/difference. We try to reject it.

H₁ (Alternative)

What you want to support — the new claim.

Example: "Does new strategy beat the 8% benchmark?"
  • H₀: μ ≤ 8% (no outperformance)
  • H₁: μ > 8% (outperforms)

Common Tests

TestUse WhenFinance Example
t-testSmall sample, σ unknownFund return vs market benchmark
z-testLarge sample (n>30), σ knownCompare stock transaction volumes
χ² (Chi-Square)Categorical relationshipsStock movement vs sector
ANOVAMeans across >2 groupsReturns of 3+ portfolios

p-Value Decision Rule

If p < α (typically 0.05) → Reject H₀ (significant result)
If p ≥ α → Fail to reject H₀
Example: p = 0.02, α = 0.05 → significant. Evidence the strategy beats the benchmark.

Type I and Type II Errors

H₀ TrueH₀ False
Reject H₀❌ Type I (α)✓ Correct
Fail to reject✓ Correct❌ Type II (β)

6️⃣ Correlation Analysis

Pearson's Correlation Coefficient (r)

The most common measure of linear association between two variables in finance. Range: −1 to +1.

r = Σ(xᵢ − x̄)(yᵢ − ȳ) / √[Σ(xᵢ − x̄)² · Σ(yᵢ − ȳ)²]
Pearson assumes a LINEAR relationship. Non-linear patterns (e.g., U-shape) can produce r ≈ 0 even when variables are strongly related — always plot the data first.

Interpreting r

rStrength
+1Perfect positive — variables move identically up
0.7 to <1Strong positive
0.3 to 0.7Moderate
0 to 0.3Weak
0No linear relationship
−1Perfect negative

Finance Applications

🎯 Diversification

Pick low-correlated assets — gains offset losses.

🛡️ Hedging

Negatively-correlated pairs (equities & bonds) protect portfolios.

📊 Portfolio Optimisation

Key input to Modern Portfolio Theory (MPT).

⚠️ Systemic Risk

In crises, correlations increase — reducing diversification benefits.

⚠️ Correlation ≠ Causation. Linear only. Sensitive to outliers. Correlations change over time.

7️⃣ Regression Analysis

Y = β₀ + β₁X + ε

Where Y = dependent (e.g., sales), X = independent (e.g., ad spend), β₀ = intercept, β₁ = slope, ε = error.

Example — Ad Spend vs Sales

Ad Spend (X, ₹k)1015202530
Sales (Y, ₹k)50658095110

Best-fit: Y = 20 + 3X. Each extra ₹1,000 ad spend → ₹3,000 extra sales. Forecast Y for X=25: 20 + 75 = ₹95,000.

Python Implementation

from sklearn.linear_model import LinearRegression import numpy as np X = np.array([10,15,20,25,30]).reshape(-1,1) Y = np.array([50,65,80,95,110]) model = LinearRegression() model.fit(X, Y) print(f"Y = {model.intercept_:.2f} + {model.coef_[0]:.2f} X") print("Predict for ₹25k spend:", model.predict([[25]])[0])

Assumptions of Linear Regression

  1. Linearity — relationship is straight-line
  2. Independence of errors — no autocorrelation
  3. Homoscedasticity — constant error variance
  4. Normal errors — residuals normally distributed
  5. No omitted variables — all relevant predictors included

Evaluating Coefficients

  • p-value < 0.05 → variable is significant
  • |t-stat| > 2 → strong impact
  • R² = % of variance explained; Adjusted R² penalises extra variables
  • VIF > 10 → multicollinearity issue

Finance Applications & Limitations

📈 Forecasting

Stock prices, revenue, indicators.

🎯 Risk Mgmt

Portfolio sensitivity (beta) to market.

🧪 Strategy Testing

Marketing ROI, innovation impact.

⚠️ Limitations

Linear only; sensitive to outliers; multicollinearity; overfitting.

📝 Summary & Exam Tips

🎯 Key Takeaways

  • Central tendency: Mean (avg), Median (mid), Mode (most freq). Median is robust to outliers.
  • Dispersion: Range, Variance, SD, IQR. SD same units as data — easiest to interpret.
  • Sharpe Ratio = (return − risk-free) / SD — risk-adjusted performance.
  • Descriptive describes; Inferential generalises from sample to population.
  • Hypothesis testing: set H₀ (null), H₁ (alt); reject H₀ if p < α (usually 0.05).
  • Tests: t-test (small/unknown σ), z-test (large/known σ), χ² (categorical), ANOVA (>2 means).
  • Type I = rejecting true H₀; Type II = failing to reject false H₀.
  • Correlation r ∈ [−1, +1]. Strong if |r| ≥ 0.7. ≠ causation.
  • Regression Y = β₀ + β₁X + ε. Assumes linearity, independence, homoscedasticity, normal errors.
  • Evaluate β: p < 0.05 significant; R² = % variance explained.
📌 Common Pitfalls:
  • Reporting Mean for skewed data — use Median instead
  • Inferring causation from correlation
  • Forgetting regression assumptions (linearity, homoscedasticity)
  • Using high R² as the only measure — Adjusted R² is better for multi-variable models
  • p-values can be misleading with small/large samples — always check effect size
