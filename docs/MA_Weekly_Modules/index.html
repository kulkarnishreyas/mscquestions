Study Strategy

Use these proven techniques to maximise your exam readiness across all 12 modules.

Active Recall

After reading each section, close it and write down the key formula or concept from memory before checking.

Spaced Repetition

Review Weeks 1–3 again after finishing Week 6. Revisit all modules 3 days before the exam.

Work the Numbers

Don't just read the case studies – re-derive the calculations yourself on paper for ANOVA, CLV, RFM, etc.

Quiz First

Jump to the quiz before reading to identify gaps. Then study those specific sections deeply.

Link Concepts

Build connections: RFM (W1) feeds Customer Analytics (W7); ANOVA (W2) links to A/B Testing (W1).

Python Practice

Run all Python code blocks in Google Colab. Modify parameters to see how outputs change.

Diagram Recall

Practice drawing key diagrams: Porter's Five Forces, Kano Model, AIDA funnel from memory.

Exam Prioritisation

High-yield: ANOVA (W2), CLV (W6), Attribution (W1), Conjoint (W4), ROMI (W9). Master these first.
